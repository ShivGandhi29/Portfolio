Detour App Showcase

React Native iOS App with Supabase Backend
Published to the App Store

Inspiration

The idea for Detour was to combine the best elements of existing popular apps to create something unique. Taking inspiration from these leading platforms:

Snapchat

Ephemeral sharing

Google Maps

Location discovery

BeReal

Authentic moments

Instagram

Visual storytelling

Spotify

UI inspiration

Detour focuses on sharing the location of cool places you've been with friends. The app creates a social map where users can discover hidden gems, share authentic experiences, and connect with friends through location-based content. The user interface draws inspiration from Spotify's clean, intuitive design and smooth user experience.

App Screenshots

Explore the Detour app through these key screenshots showcasing the user interface and functionality.

Home Screen

The main dashboard showing personalized content, location-based posts, and quick access to key features. Users can filter posts by categories (cafes, activities, events, etc.) to discover exactly what they're looking for. Clean, intuitive design with smooth navigation.

Social Features

Connect with friends, join groups, and share location-based content. Filter posts by categories to find specific types of experiences. Real-time updates and social interactions powered by Supabase.

Beacons

Beacons is a way to send location based invites for spontaneous meetups to your friends.

Key Features & Technologies

Built with modern technologies and best practices for mobile development.

Frontend Technologies

  • React Native with Expo Framework
  • TypeScript for type safety
  • Native iOS development with Xcode
  • Responsive UI/UX design

Backend & Services

  • Supabase PostgreSQL database
  • Supabase Authentication
  • Edge Functions for serverless logic
  • Real-time subscriptions

Core Features

  • User authentication & profiles
  • Location-based content discovery
  • Social features (friends, groups)
  • Real-time messaging & updates

Monetization

  • Google AdMob integration
  • RevenueCat paywall system
  • In-app purchase support
  • Subscription management

Development Process

From concept to App Store publication - a comprehensive development journey.

Planning & Design

User research, wireframing, and UI/UX design. Created user personas and mapped user journeys for optimal experience.

Development

Agile development with 4-person team. Regular sprints, code reviews, and continuous integration for quality assurance.

Launch & Iteration

App Store submission, user feedback collection, and iterative improvements based on real user data and analytics.
